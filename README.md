# React Frontend Tasks Rija Arshad
This repository contains two frontend tasks built using React.
## Task 1  Product Listing Dashboard
Features:
- API data fetching (DummyJSON)
- Search with debounce
- Sorting (price & name)
- Pagination
- Loading skeletons
- Error handling
- Responsive UI
Run locally:
cd task1-listing
npm install
npm run dev
## Task 2  Form Validation
Features:
- Full form validation
- Inline error messages
- Password strength meter
- Success screen
- Responsive design
Run locally:
cd task2-form
npm install
npm run dev
